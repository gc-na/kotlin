# Kotlin의 while 문: 반복문 사용법과 예제

## 개요
Kotlin에서 `while` 문은 주어진 조건이 참인 동안 코드를 반복 실행하는 제어 구조입니다. 이를 통해 특정 작업을 반복적으로 수행할 수 있습니다.

## 문서화
Kotlin의 `while` 문은 반복문 중 하나로, 조건이 참인 동안 블록 내의 코드를 실행합니다. 다음은 `while` 문 사용 방법에 대한 자세한 설명입니다.

### 목적
`while` 문은 반복적으로 작업을 수행할 필요가 있을 때 유용합니다. 조건이 변할 때까지 코드를 계속 실행할 수 있습니다.

### 사용법
`while` 문의 기본 구문은 다음과 같습니다:

```kotlin
while (조건) {
    // 실행할 코드
}
```

- **조건**: Boolean 값을 반환하는 표현식으로, 참일 경우 블록 내의 코드가 실행됩니다.
- **실행할 코드**: 조건이 참인 동안 반복 실행될 코드 블록입니다.

### 세부 사항
- 조건이 처음부터 거짓이면 코드 블록은 단 한 번도 실행되지 않습니다.
- `while` 문은 무한 루프를 유발할 수 있으므로 주의해야 합니다. 조건이 항상 참으로 평가될 경우, 프로그램이 종료되지 않고 계속 실행됩니다.

## 예제
### 기본 사용 예제
```kotlin
var count = 1
while (count <= 5) {
    println("카운트: $count")
    count++
}
```
위의 예제는 1부터 5까지의 숫자를 출력합니다.

### 무한 루프 예제
```kotlin
var number = 1
while (true) {
    println("무한 반복: $number")
    number++
    if (number > 5) break  // 조건을 추가하여 무한 루프 방지
}
```
이 예제는 `true`로 인해 무한 루프를 생성하지만, 조건문을 통해 멈추게 됩니다.

## 설명
`while` 문을 사용할 때의 일반적인 주의 사항은 다음과 같습니다:
- **무한 루프**: 조건이 항상 참인 경우, 프로그램이 멈추지 않고 계속 실행됩니다. 이를 방지하기 위해 반드시 조건을 업데이트해야 합니다.
- **조건식**: 조건식이 Boolean 값을 반환해야 하며, 잘못된 표현식은 컴파일 오류를 발생시킵니다.
- **성능**: `while` 문은 빠르게 반복할 수 있지만, 조건이 복잡할 경우 성능에 영향을 줄 수 있습니다.

## 한줄 요약
Kotlin의 `while` 문은 주어진 조건이 참일 때 코드 블록을 반복 실행하는 강력한 제어 구조입니다.
