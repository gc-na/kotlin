# استخدام الكلمة المفتاحية "in" في لغة كوتلن

## ملخص
تُستخدم الكلمة المفتاحية "in" في لغة كوتلن لتحديد وجود عنصر داخل مجموعة أو للتكرار عبر العناصر في مجموعة معينة. تُعتبر "in" جزءًا أساسيًا في كتابة الشيفرة بطريقة أكثر وضوحًا وسهولة.

## الوثائق
تُستخدم الكلمة المفتاحية "in" في كوتلن في سياقات متعددة، منها:

1. **التحقق من وجود عنصر في مجموعة**: حيث تُستخدم للتحقق مما إذا كان عنصر معين موجودًا داخل مجموعة مثل قائمة أو مجموعة أو مصفوفة.
2. **التكرار عبر العناصر**: تُستخدم "in" في حلقات `for` للتكرار عبر العناصر في مجموعة.

### الاستخدام
- **التحقق من وجود عنصر في مجموعة**:
  ```kotlin
  val numbers = listOf(1, 2, 3, 4, 5)
  val containsThree = 3 in numbers // يعيد true
  ```

- **التكرار عبر العناصر**:
  ```kotlin
  for (number in numbers) {
      println(number)
  }
  ```

## الأمثلة
### مثال 1: التحقق من وجود عنصر
```kotlin
val fruits = listOf("تفاح", "موز", "برتقال")
if ("موز" in fruits) {
    println("الموز موجود في القائمة.")
}
```

### مثال 2: استخدام حلقة for للتكرار
```kotlin
val letters = 'a'..'e' // نطاق من الأحرف
for (letter in letters) {
    println(letter)
}
```

## الشرح
### الأخطاء الشائعة
- **عدم التحقق من النوع**: عند استخدام "in" للتحقق من وجود عنصر، تأكد من أن العنصر من نفس النوع الموجود في المجموعة. 
- **استخدام "in" بشكل غير صحيح في النطاقات**: تأكد من أن النطاق معرف بشكل صحيح، وإلا قد تظهر أخطاء عند التنفيذ.

### ملاحظات إضافية
- "in" يمكن استخدامها مع أنواع بيانات مختلفة مثل القوائم والمجموعات والنطاقات.
- استخدام "in" يُسهل قراءة الشيفرة ويجعلها أكثر وضوحًا مقارنة بالعمليات التقليدية للتحقق من وجود عناصر.

## ملخص جملة واحدة
تُستخدم الكلمة المفتاحية "in" في كوتلن للتحقق من وجود عنصر في مجموعة أو للتكرار عبر عناصر مجموعة معينة.
