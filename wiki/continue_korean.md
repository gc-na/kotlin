# Kotlin의 continue 문: 루프 제어를 위한 효율적인 도구

## 요약
Kotlin의 `continue` 문은 루프 내에서 특정 조건이 충족되었을 때 현재 반복을 건너뛰고 다음 반복으로 넘어가는 제어문입니다. 이를 통해 코드의 가독성을 높이고 불필요한 실행을 줄일 수 있습니다.

## 문서화
`continue` 문은 반복문에서 특정 조건을 만족할 때 현재 반복을 종료하고 다음 반복으로 진행하도록 도와주는 기능입니다. 이 문장은 `for`, `while`, `do...while` 루프와 함께 사용될 수 있습니다. `continue`는 조건문과 함께 사용되어 특정 상황에서 반복을 건너뛰고 코드 실행을 최적화하는 데 유용합니다.

### 사용법
- `continue` 문은 루프 내에서 특정 조건을 평가한 후, 그 조건이 참일 경우 실행됩니다.
- 사용 예:
  ```kotlin
  for (i in 1..10) {
      if (i % 2 == 0) continue // 짝수인 경우 현재 반복을 건너뛰기
      println(i) // 홀수만 출력
  }
  ```

## 예제
### 기본 사용 예제
```kotlin
fun main() {
    for (i in 1..10) {
        if (i % 3 == 0) continue // 3의 배수인 경우 건너뛰기
        println(i) // 1, 2, 4, 5, 7, 8, 10 출력
    }
}
```

### while 루프에서의 사용
```kotlin
fun main() {
    var i = 1
    while (i <= 10) {
        i++
        if (i % 2 == 0) continue // 짝수인 경우 건너뛰기
        println(i) // 1, 3, 5, 7, 9 출력
    }
}
```

## 설명
`continue` 문은 루프 내에서 조건을 평가하여 특정 조건이 참일 경우, 해당 조건 이후의 코드를 건너뛰고 루프의 다음 반복으로 넘어갑니다. 이는 코드의 가독성을 높이고, 특정 조건을 만족하는 항목을 쉽게 처리할 수 있게 도와줍니다.

### 일반적인 함정 및 주의사항
- `continue` 문은 루프 내에서만 사용할 수 있으며, 루프 바깥에서는 사용할 수 없습니다.
- 조건문이 항상 참이 될 경우, 무한 루프를 초래할 수 있으므로 주의가 필요합니다.
- `continue` 문은 코드의 흐름을 변경하기 때문에, 코드의 논리를 이해하는 데 어려움을 줄 수 있습니다. 따라서 필요한 경우에만 사용하도록 합니다.
