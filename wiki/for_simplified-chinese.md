# Kotlin 中的 for 循环使用指南

## 摘要
在 Kotlin 中，`for` 循环是一种用于遍历集合、数组或区间的控制结构，能够简洁地执行重复操作。

## 文档
`for` 循环在 Kotlin 中用于遍历集合、数组或区间。它的语法简洁明了，能够提高代码的可读性和可维护性。基本的语法格式如下：

```kotlin
for (item in collection) {
    // 对每个 item 执行的操作
}
```

### 目的
`for` 循环的主要目的是简化对元素的遍历，使得开发者可以更直观地操作集合数据。

### 用法
- **遍历数组**：可以直接遍历数组中的元素。
- **遍历集合**：可以遍历列表、集合等实现了 Iterable 接口的对象。
- **遍历区间**：可以遍历一个整数区间，常用于循环计数。

### 细节
- `for` 循环会自动处理集合中的元素类型，开发者无需显式指定类型。
- 可以使用 `indices` 属性获取数组或集合的索引范围。

## 示例
### 遍历数组
```kotlin
val numbers = arrayOf(1, 2, 3, 4, 5)
for (number in numbers) {
    println(number)
}
```

### 遍历集合
```kotlin
val fruits = listOf("苹果", "香蕉", "橙子")
for (fruit in fruits) {
    println(fruit)
}
```

### 遍历区间
```kotlin
for (i in 1..5) {
    println(i)
}
```

### 使用步长
```kotlin
for (i in 1..10 step 2) {
    println(i) // 打印 1, 3, 5, 7, 9
}
```

## 解释
- **常见陷阱**：使用 `for` 循环时，确保集合或数组不为空，以避免遍历时发生空指针异常。
- **可变集合**：当遍历可变集合时，如果在循环中修改集合，可能会导致意外的行为或并发问题。
- **闭区间与开区间**：注意 `..` 表达式是闭区间，意味着包含结束值，而 `until` 表达式是开区间，不包含结束值。

## 一句话总结
Kotlin 中的 `for` 循环是一种简洁高效的方式，用于遍历集合、数组和区间。
