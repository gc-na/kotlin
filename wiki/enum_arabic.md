# استخدام enum في لغة كوتلن: دليل شامل

## ملخص
تُعتبر "enum" في كوتلن نوعًا خاصًا من البيانات يُستخدم لتعريف مجموعة من القيم الثابتة. يساهم استخدامها في تحسين تنظيم الشيفرة البرمجية وزيادة وضوحها.

## الوثائق
تُستخدم الكلمات المحجوزة `enum class` لتعريف نوع enum في كوتلن. يتيح لك هذا النوع تعريف مجموعة من الثوابت المرتبطة ببعضها. على سبيل المثال، يمكن استخدام enum لتعريف أيام الأسبوع أو أنواع السيارات.

### الغرض
الغرض الأساسي من `enum` هو تحسين تنظيم الشيفرة وجعلها أكثر وضوحًا وسهولة في القراءة والصيانة. 

### الاستخدام
لتعريف enum، يمكنك استخدام الكود التالي:

```kotlin
enum class Day {
    SUNDAY, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY
}
```

يمكنك استخدام enum في الشيفرة كما يلي:

```kotlin
fun printDay(day: Day) {
    when (day) {
        Day.SUNDAY -> println("اليوم هو الأحد")
        Day.MONDAY -> println("اليوم هو الإثنين")
        // يمكنك إضافة المزيد من الحالات هنا
        else -> println("يوم غير معروف")
    }
}
```

## الأمثلة
### مثال 1: تعريف واستخدام enum

```kotlin
enum class Color {
    RED, GREEN, BLUE
}

fun getColorMeaning(color: Color): String {
    return when (color) {
        Color.RED -> "لون أحمر"
        Color.GREEN -> "لون أخضر"
        Color.BLUE -> "لون أزرق"
    }
}

fun main() {
    val myColor = Color.RED
    println(getColorMeaning(myColor)) // سيطبع: لون أحمر
}
```

### مثال 2: إضافة خصائص للـ enum

```kotlin
enum class Planet(val distanceFromSun: Int) {
    MERCURY(57_910_000),
    VENUS(108_200_000),
    EARTH(149_600_000),
    MARS(227_900_000);
}

fun main() {
    println("المسافة من الشمس لكوكب الأرض: ${Planet.EARTH.distanceFromSun} كيلومتر")
}
```

## الشرح
### الأخطاء الشائعة
- **عدم استخدام `when` مع جميع القيم:** عند استخدام `when` مع enum، تأكد من تغطية جميع القيم أو إضافة حالة `else` لتفادي الأخطاء.
- **تغيير قيمة enum:** القيم في enum ثابتة ولا يمكن تعديلها بعد تعريفها. تأكد أنك تفهم أن enum مناسب للقيم الثابتة فقط.

### ملاحظات إضافية
- يمكن أن تحتوي enums على خصائص ودوال، مما يجعلها أكثر مرونة.
- يُفضل استخدام enums عند الحاجة إلى مجموعة من القيم الثابتة ذات الصلة، مما يسهل قراءة الشيفرة وفهمها.

## ملخص جملة واحدة
تُستخدم enums في كوتلن لتعريف مجموعة من القيم الثابتة، مما يُحسن تنظيم الشيفرة ووضوحها.
