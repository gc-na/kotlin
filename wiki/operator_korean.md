# Kotlin에서의 연산자(Operator): 기능과 활용

## 개요
Kotlin에서 연산자는 데이터 타입 간의 연산을 수행하는 특별한 기호입니다. 이들은 기본적인 수학 연산은 물론, 객체 간의 비교 및 조작을 가능하게 합니다. 이 문서에서는 Kotlin에서 사용되는 다양한 연산자에 대해 설명합니다.

## 문서화

### 목적
Kotlin의 연산자는 코드의 가독성을 높이고, 개발자가 직관적으로 데이터를 조작할 수 있게 합니다. 이러한 연산자는 클래스에 정의할 수 있으며, 기존의 연산자를 오버로딩하여 사용자 정의 객체에 대한 연산을 수행할 수 있습니다.

### 사용법
Kotlin에서 연산자는 다음과 같은 여러 종류로 나뉩니다:
- **산술 연산자**: `+`, `-`, `*`, `/`, `%`
- **비교 연산자**: `==`, `!=`, `<`, `>`, `<=`, `>=`
- **논리 연산자**: `&&`, `||`, `!`
- **대입 연산자**: `=`, `+=`, `-=`, `*=`, `/=`, `%=`
- **인덱스 연산자**: `[]`
- **호출 연산자**: `()`

이외에도, 사용자 정의 클래스를 위해 연산자를 오버로딩할 수 있습니다.

### 세부 사항
연산자를 오버로딩하려면, 클래스 내에서 특정 메서드를 정의해야 합니다. 예를 들어, `plus` 메서드는 `+` 연산자를 오버로딩하는 데 사용됩니다.

```kotlin
data class Vector(val x: Int, val y: Int) {
    operator fun plus(other: Vector): Vector {
        return Vector(x + other.x, y + other.y)
    }
}
```

## 예제

### 기본 사용 예제

1. **산술 연산자**

```kotlin
val a = 10
val b = 5
val sum = a + b // 결과: 15
```

2. **비교 연산자**

```kotlin
val isEqual = (a == b) // 결과: false
```

3. **논리 연산자**

```kotlin
val isTrue = (a > b) && (b > 0) // 결과: true
```

4. **사용자 정의 연산자 오버로딩**

```kotlin
val vector1 = Vector(2, 3)
val vector2 = Vector(4, 5)
val resultVector = vector1 + vector2 // 결과: Vector(x=6, y=8)
```

## 설명
Kotlin에서 연산자를 사용할 때 주의할 점은 오버로딩 시 메서드의 이름이 고정되어 있다는 것입니다. 또한, 잘못된 오버로딩은 코드의 가독성을 해칠 수 있습니다. 예를 들어, `plus` 메서드를 잘못 구현하면 예기치 않은 결과를 초래할 수 있습니다. 따라서 오버로딩 시에는 명확한 의도를 가지고 구현해야 합니다.

## 한 줄 요약
Kotlin의 연산자는 데이터 조작을 위한 기호로, 기본 연산뿐 아니라 사용자 정의 객체에 대한 연산도 지원합니다.
