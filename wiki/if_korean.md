# Kotlin의 "if" 문: 조건문 활용하기

## 개요
Kotlin에서 "if" 문은 조건에 따라 코드 블록의 실행 여부를 결정하는 제어 구조입니다. 이 기능은 프로그래밍에서 조건적 로직을 구현하는 데 필수적입니다.

## 문서화
Kotlin의 "if" 문은 조건이 참(true)인지 거짓(false)인지에 따라 서로 다른 코드를 실행할 수 있도록 해줍니다. "if" 문은 다음과 같은 방식으로 사용됩니다:

```kotlin
if (조건) {
    // 조건이 참일 때 실행할 코드
} else {
    // 조건이 거짓일 때 실행할 코드
}
```

또한, "if" 문은 표현식으로 사용될 수 있어 값을 반환할 수 있다는 점이 특징입니다:

```kotlin
val 결과 = if (조건) {
    // 조건이 참일 때 반환할 값
} else {
    // 조건이 거짓일 때 반환할 값
}
```

## 예제
1. 기본 "if" 문 사용:

```kotlin
val a = 10
if (a > 5) {
    println("a는 5보다 큽니다.")
} else {
    println("a는 5보다 작습니다.")
}
```

2. "if" 문을 표현식으로 사용:

```kotlin
val max = if (a > 5) a else 5
println("최대값: $max")
```

3. 여러 조건을 처리하는 "if" 문:

```kotlin
val b = 20
if (a > b) {
    println("a는 b보다 큽니다.")
} else if (a < b) {
    println("a는 b보다 작습니다.")
} else {
    println("a와 b는 같습니다.")
}
```

## 설명
- **조건식**: "if" 문에서 사용되는 조건식은 Boolean 값을 반환해야 합니다.
- **블록**: "if"와 "else" 뒤에 오는 코드 블록은 중괄호 `{}`로 감싸거나 단일 문장일 경우 생략할 수 있습니다.
- **가독성**: 여러 조건을 처리할 때는 `when` 문을 사용하는 것이 코드의 가독성을 높일 수 있습니다.
- **중첩 사용**: "if" 문은 서로 중첩하여 사용할 수 있지만, 지나치게 복잡해질 수 있으므로 주의가 필요합니다.

## 한 줄 요약
Kotlin에서 "if" 문은 조건에 따라 코드 실행을 제어하는 강력한 도구입니다.
