# Kotlin의 break 명령어: 사용법과 예제

## 개요
Kotlin에서 `break` 명령어는 반복문을 조기에 종료하는 데 사용됩니다. 이 명령어는 특정 조건이 충족될 때 반복문을 중단하고, 반복문 이후의 코드로 제어를 전환합니다.

## 문서화
### 목적
`break`는 주로 `for`, `while` 및 `do-while` 반복문 내부에서 사용되며, 반복문을 즉시 종료할 수 있게 해줍니다. 이는 반복문 내에서 특정 조건을 만족할 때 반복의 흐름을 제어할 수 있는 유용한 기능입니다.

### 사용법
`break` 명령어는 다음과 같은 형식으로 사용됩니다:

```kotlin
for (i in 1..10) {
    if (i == 5) {
        break
    }
    println(i)
}
```

이 예제에서 `i`가 5와 같아질 때 `break`가 실행되어 반복문이 종료됩니다.

### 세부 사항
- `break`는 중첩된 반복문에서도 사용할 수 있으며, 이를 통해 가장 가까운 반복문을 종료합니다.
- `break`는 레이블과 함께 사용할 수 있어, 특정 레이블이 붙은 반복문을 종료할 수도 있습니다.
- Kotlin에서는 `break`와 `continue`를 사용하여 반복문의 흐름을 더욱 세밀하게 제어할 수 있습니다.

## 예제
1. **기본 사용 예제**:
```kotlin
fun main() {
    for (i in 1..10) {
        if (i == 6) {
            break
        }
        println(i) // 출력: 1, 2, 3, 4, 5
    }
}
```

2. **중첩된 반복문에서의 사용**:
```kotlin
fun main() {
    outerLoop@ for (i in 1..5) {
        for (j in 1..5) {
            if (j == 3) {
                break@outerLoop
            }
            println("i: $i, j: $j")
        }
    }
}
```

이 예제에서는 `j`가 3일 때 외부 반복문이 종료됩니다.

## 설명
`break` 명령어를 사용할 때 주의해야 할 몇 가지 사항이 있습니다:
- `break`는 반복문 내의 조건문과 함께 사용될 때 가장 효과적입니다. 조건문이 없으면 반복문은 끝까지 실행됩니다.
- 중첩된 반복문에서는 특정 레이블을 사용하여 원하는 반복문을 선택적으로 종료할 수 있습니다. 이를 통해 코드의 가독성을 높이고 불필요한 반복을 방지할 수 있습니다.
- `break`를 남용하면 코드의 흐름이 복잡해질 수 있으므로, 적절한 상황에서만 사용하는 것이 좋습니다.

## 한 문장 요약
Kotlin의 `break` 명령어는 반복문을 조기에 종료할 수 있도록 하는 강력한 도구입니다.
