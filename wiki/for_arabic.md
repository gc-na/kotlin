# استخدام جملة "for" في لغة كوتلن: كيفية تكرار العناصر بكفاءة

## ملخص
تعتبر جملة "for" في كوتلن من الأدوات الأساسية التي تُستخدم للتكرار على عناصر مجموعة معينة. توفر هذه الجملة طريقة بسيطة وسهلة للتنقل خلال العناصر وتطبيق عمليات معينة عليها.

## الوثائق
تُستخدم جملة "for" في كوتلن لتكرار العناصر في المجموعات مثل القوائم، والمصفوفات، والنطاقات. يمكن استخدامها بطرق متعددة، مما يجعلها أداة قوية في كتابة الشيفرة.

### الاستخدام
يمكن كتابة جملة "for" بشكلين رئيسيين:
1. **التكرار عبر النطاقات**: 
   ```kotlin
   for (i in 1..5) {
       println(i)
   }
   ```
   في هذا المثال، سيتم طباعة الأرقام من 1 إلى 5.

2. **التكرار عبر المجموعات**: 
   ```kotlin
   val items = listOf("apple", "banana", "cherry")
   for (item in items) {
       println(item)
   }
   ```
   هنا، سيتم طباعة كل عنصر من عناصر القائمة.

### تفاصيل إضافية
- **النطاقات**: يمكن استخدام `until` لإنشاء نطاق غير شامل للنهاية:
  ```kotlin
  for (i in 1 until 5) {
      println(i) // سيطبع الأرقام من 1 إلى 4
  }
  ```

- **التكرار العكسي**: يمكنك استخدام `downTo` للتكرار من قيمة أعلى إلى قيمة أقل:
  ```kotlin
  for (i in 5 downTo 1) {
      println(i)
  }
  ```

## أمثلة
### مثال 1: تكرار عبر مصفوفة
```kotlin
val numbers = arrayOf(1, 2, 3, 4, 5)
for (number in numbers) {
    println(number)
}
```

### مثال 2: استخدام `withIndex`
```kotlin
val fruits = listOf("apple", "banana", "cherry")
for ((index, fruit) in fruits.withIndex()) {
    println("$index: $fruit")
}
```

## الشرح
### الأخطاء الشائعة
- **عدم الانتباه لنطاق التكرار**: تأكد من استخدام النطاقات بشكل صحيح، حيث قد يؤدي استخدام نطاق خاطئ إلى عدم تكرار جميع العناصر أو تكرار العناصر غير المرغوبة.
- **عدم استخدام `break` و `continue`**: يمكنك التحكم في تدفق التكرار باستخدام `break` للخروج من الحلقة أو `continue` لتجاوز بعض التكرارات.

### ملاحظات إضافية
- يمكن استخدام جملة "for" مع أي نوع من مجموعات البيانات في كوتلن، مما يجعلها مرنة وسهلة الاستخدام.
- تذكر أن الأداء قد يتأثر بناءً على نوع المجموعة التي تتعامل معها، لذا يجب مراعاة ذلك عند كتابة الشيفرة.

## ملخص جملة واحدة
تعتبر جملة "for" في كوتلن أداة فعالة لتكرار العناصر في المجموعات والنطاقات، مما يسهل كتابة الشيفرة بشكل أكثر كفاءة.
