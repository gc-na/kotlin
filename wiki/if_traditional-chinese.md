# Kotlin 的 if 陳述式：條件控制的強大工具

## 概要
在 Kotlin 中，`if` 陳述式用於條件控制，能夠根據布林值的結果執行不同的程式碼區塊。它不僅可以用作控制流語句，還可以作為一個表達式，直接返回值。

## 文檔
`if` 陳述式是 Kotlin 中最基本且常用的控制流結構之一。它的功能類似於其他編程語言中的 `if` 陳述式，但 Kotlin 的 `if` 除了可以執行邏輯判斷外，還可以直接返回值，這使得代碼更加簡潔。

### 用法
`if` 陳述式的基本語法如下：

```kotlin
if (條件) {
    // 當條件為 true 時執行的程式碼
} else {
    // 當條件為 false 時執行的程式碼
}
```

此外，Kotlin 允許使用 `if` 作為表達式，這意味著你可以將 `if` 陳述式的結果賦值給變量：

```kotlin
val max = if (a > b) a else b
```

### 詳細說明
- **多重條件**：可以使用 `else if` 來處理多個條件：
  
```kotlin
if (條件1) {
    // 條件1為真時執行的程式碼
} else if (條件2) {
    // 條件2為真時執行的程式碼
} else {
    // 以上條件均為假時執行的程式碼
}
```

- **省略 else 區塊**：如果只需要處理一個條件，`else` 區塊是可選的。

## 範例
以下是一些 `if` 陳述式的基本用法範例：

### 範例 1：基本的 if 陳述式
```kotlin
val number = 10
if (number > 5) {
    println("數字大於 5")
} else {
    println("數字小於或等於 5")
}
```

### 範例 2：使用 if 作為表達式
```kotlin
val a = 10
val b = 20
val max = if (a > b) a else b
println("最大值是: $max")
```

### 範例 3：多重條件
```kotlin
val score = 85
val grade = if (score >= 90) {
    "A"
} else if (score >= 80) {
    "B"
} else if (score >= 70) {
    "C"
} else {
    "D"
}
println("你的成績是: $grade")
```

## 說明
在使用 `if` 陳述式時，有幾個常見的陷阱和注意事項：

- **布林表達式**：確保條件表達式的結果為布林型別 (Boolean)，否則會導致編譯錯誤。
- **用作表達式時的類型一致性**：當使用 `if` 作為表達式時，所有的條件分支必須返回相同的類型，否則會出現類型不匹配的錯誤。
- **沒有 else 的情況**：如果省略 `else`，並且所有條件均不為真，則不會有任何值返回，這在某些情況下可能造成未預期的行為。

## 單行摘要
Kotlin 的 `if` 陳述式是一種強大的條件控制工具，能夠根據條件執行不同的程式碼或返回值。
