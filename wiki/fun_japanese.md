# Kotlinにおける「fun」キーワードの徹底ガイド

## 概要
Kotlinプログラミング言語における「fun」キーワードは、関数を定義するために使用されます。このキーワードは、Kotlinの関数型プログラミングの特徴を活かす重要な要素です。

## ドキュメント
### 目的
「fun」キーワードは、Kotlinで関数を定義する際に必要です。関数は一連の処理を実行し、値を返すことができます。

### 使用法
関数を定義するには、次の構文を使用します：

```kotlin
fun 関数名(引数名: 引数の型): 戻り値の型 {
    // 関数の処理
}
```

### 詳細
- **関数名**: 関数を識別するための名前です。小文字で始め、キャメルケースを用いることが推奨されています。
- **引数**: 関数に渡されるデータ。複数の引数を持つことも可能です。
- **戻り値の型**: 関数が返す値の型。戻り値がない場合は`Unit`を指定します。

## 例
以下は、基本的な関数の使い方の例です。

```kotlin
fun greet(name: String): String {
    return "こんにちは、$name!"
}

fun main() {
    val message = greet("太郎")
    println(message)  // 出力: こんにちは、太郎!
}
```

## 説明
- **引数の型**: Kotlinでは、引数に型を明示する必要があります。型を指定しない場合、コンパイラはエラーを返します。
- **戻り値の型**: 関数の戻り値の型は、必ず定義する必要があります。ただし、戻り値がない場合は`Unit`を省略することができます。
- **スコープ**: 関数内で定義された変数は、その関数のスコープ内でのみ有効です。他の関数や外部からはアクセスできません。

## 一行要約
Kotlinの「fun」キーワードは、関数を定義するために使用され、関数型プログラミングの基盤を提供します。
