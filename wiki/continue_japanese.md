# Kotlinにおける「continue」: 繰り返し処理の制御

## 概要
Kotlinの「continue」文は、ループ内での処理を制御するためのキーワードであり、現在の反復をスキップして次の反復に移るために使用されます。これにより、特定の条件を満たしていない場合に、残りのコードを実行せずにループの次のサイクルに進むことができます。

## ドキュメント
「continue」は、for、while、do-while ループ内で使用され、特定の条件を評価し、その条件が満たされた場合にループの次の反復を開始します。これにより、無駄な処理を避け、効率的なプログラムを書くことができます。

### 使用方法
以下の構文で「continue」を使用します：

```kotlin
for (i in 1..10) {
    if (i % 2 == 0) {
        continue // iが偶数の場合、次の反復に移る
    }
    println(i) // iが奇数の場合のみ出力
}
```

この例では、1から10までの数字の中で偶数をスキップし、奇数だけを出力します。

## 例
### 基本的な使用例
1. 偶数をスキップする例：

```kotlin
for (i in 1..10) {
    if (i % 2 == 0) {
        continue
    }
    println(i) // 1, 3, 5, 7, 9が出力される
}
```

2. 特定の条件でスキップする例：

```kotlin
val numbers = listOf(1, 2, 3, 4, 5)
for (number in numbers) {
    if (number == 3) {
        continue
    }
    println(number) // 1, 2, 4, 5が出力される
}
```

## 説明
「continue」を使用する際の一般的な落とし穴として、条件式が適切に設定されていない場合、意図しない挙動を引き起こすことがあります。また、ネストされたループ内で使用する場合、どのループが「continue」の影響を受けるかを理解しておくことが重要です。内側のループで「continue」を使用すると、外側のループには影響を与えず、内側のループの次の反復に移ることになります。

### 注意点
- 「continue」は、使用するループの種類に応じて異なる動作をすることがあります。特に、ネストされたループでは、意図したループに対して正しく動作するか確認してください。

## 一文要約
Kotlinの「continue」文は、ループ内で条件を満たさない場合に現在の反復をスキップし、次の反復に進むためのキーワードです。
