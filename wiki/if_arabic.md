# استخدام جملة "if" في لغة كوتلن: دليل شامل

## ملخص
تعتبر جملة "if" في كوتلن من البنى الأساسية المستخدمة للتحكم في تدفق البرنامج، حيث تتيح للمستخدمين اتخاذ قرارات بناءً على شروط معينة.

## الوثائق
تستخدم جملة "if" في كوتلن لتقييم شرط وإجراء عملية معينة بناءً على نتيجة ذلك الشرط. يمكن استخدامها كعبارة شرطية بسيطة أو كعبارة تعبيرية للرجوع إلى قيمة. التركيب العام لجملة "if" هو كالتالي:

```kotlin
if (شرط) {
    // الكود الذي سيتم تنفيذه إذا كان الشرط صحيحًا
} else {
    // الكود الذي سيتم تنفيذه إذا كان الشرط غير صحيح
}
```

بالإضافة إلى ذلك، يمكن استخدام "if" بدون جملة "else" إذا كان الشروط تتطلب ذلك:

```kotlin
if (شرط) {
    // الكود الذي سيتم تنفيذه إذا كان الشرط صحيحًا
}
```

يمكن أيضًا استخدام جملة "if" كعبارة تعبيرية، مما يعني أنها يمكن أن تعود بقيمة:

```kotlin
val نتيجة = if (شرط) {
    القيمة_إذا_كان_الشرط_صحيحًا
} else {
    القيمة_إذا_كان_الشرط_غير_صحيح
}
```

## الأمثلة
### مثال بسيط
```kotlin
fun main() {
    val العدد = 10
    if (عدد > 0) {
        println("العدد موجب")
    } else {
        println("العدد غير موجب")
    }
}
```

### مثال على استخدام "if" كعبارة تعبيرية
```kotlin
fun main() {
    val عدد = 5
    val نوع = if (عدد % 2 == 0) {
        "زوجي"
    } else {
        "فردي"
    }
    println("العدد هو: $نوع")
}
```

## الشرح
من الأمور المهمة في استخدام جملة "if" في كوتلن:

- **عدم وجود قوسين**: في حالة وجود جملة شرطية واحدة فقط، يمكن الاستغناء عن الأقواس المعقوفة.
- **الإجبار على استخدام "else"**: لا يُعتبر استخدام "else" إلزاميًا، لكن استخدامه يمكن أن يساعد في جعل الكود أكثر وضوحًا.
- **التقييم المتسلسل**: يمكن استخدام جملة "if" متعددة الشروط عن طريق استخدام "else if":

```kotlin
if (شرط1) {
    // كود
} else if (شرط2) {
    // كود آخر
} else {
    // كود بديل
}
```

## ملخص جملة واحدة
تستخدم جملة "if" في كوتلن لتقييم الشروط واتخاذ قرارات في تدفق البرنامج، مما يسهل التحكم في كيفية تنفيذ الكود.
