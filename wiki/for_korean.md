# Kotlin의 for 루프: 반복문을 활용한 효과적인 코드 작성

## 개요
Kotlin의 `for` 루프는 컬렉션, 배열, 범위 등을 순회하며 반복 작업을 수행할 수 있는 유용한 기능입니다. 이 문서는 Kotlin에서 `for` 루프의 사용법, 예제, 주의할 점을 다룹니다.

## 문서
Kotlin의 `for` 루프는 주어진 요소들을 반복적으로 처리할 수 있는 구조입니다. `for` 루프는 주로 배열, 리스트, 범위 객체를 순회할 때 사용됩니다. 이를 통해 개발자는 코드의 반복 작업을 간편하게 처리할 수 있으며, 가독성을 높이고 코드의 길이를 줄일 수 있습니다.

### 사용법
Kotlin에서 `for` 루프는 다음과 같은 형식으로 사용됩니다:

```kotlin
for (item in collection) {
    // item에 대한 작업 수행
}
```

여기서 `collection`은 순회할 대상(예: 배열, 리스트, 범위 등)입니다.

## 예제
### 1. 배열 순회
```kotlin
val fruits = arrayOf("사과", "바나나", "체리")
for (fruit in fruits) {
    println(fruit)
}
```
이 코드는 배열 `fruits`의 각 요소를 출력합니다.

### 2. 리스트 순회
```kotlin
val numbers = listOf(1, 2, 3, 4, 5)
for (number in numbers) {
    println(number * 2)
}
```
이 코드는 리스트 `numbers`의 각 요소를 두 배로 만들어 출력합니다.

### 3. 범위 순회
```kotlin
for (i in 1..5) {
    println("숫자: $i")
}
```
이 코드는 1부터 5까지의 숫자를 출력합니다.

## 설명
Kotlin의 `for` 루프 사용 시 주의할 점은 다음과 같습니다:

1. **변경 불가능한 컬렉션**: `for` 루프를 사용하여 컬렉션의 요소를 순회할 때, 컬렉션의 크기를 변경하면 예기치 않은 결과가 발생할 수 있습니다. 따라서 순회 중에는 컬렉션을 변경하지 않도록 주의해야 합니다.
   
2. **타입 안전성**: Kotlin은 강력한 타입 시스템을 제공하므로, `for` 루프에서 사용하는 컬렉션의 타입이 일치하지 않으면 컴파일 오류가 발생합니다.

3. **인덱스 접근**: `for` 루프는 인덱스를 사용하여 요소에 접근할 수 없습니다. 인덱스를 통해 접근해야 할 경우, `indices` 속성을 활용해야 합니다.

## 한 줄 요약
Kotlin의 `for` 루프는 배열, 리스트, 범위 등을 순회하며 반복 작업을 간편하게 처리할 수 있는 강력한 도구입니다.
