# استخدام الكلمة المفتاحية "return" في كوتلن: الدليل الشامل

## الملخص
تعتبر الكلمة المفتاحية "return" في كوتلن أداة أساسية تستخدم لإنهاء دالة وإرجاع قيمة منها. يسهل استخدامها كتابة برامج أكثر وضوحًا وتنظيمًا.

## الوثائق
تُستخدم الكلمة المفتاحية "return" في كوتلن لإنهاء تنفيذ دالة وإرجاع قيمة للمستدعي. يمكن استخدامها في جميع أنواع الدوال، سواء كانت دوال عادية أو دوال من نوع "Lambda". عند استدعاء دالة تحتوي على "return"، يتم إنهاء الدالة على الفور وإرجاع القيمة المحددة.

### الاستخدام
- **العودة من دالة عادية**: في الدوال العادية، يمكن استخدام "return" لإرجاع قيمة من نوع الدالة. إذا كانت الدالة لا تُرجع قيمة، يمكن استخدام "return" بدون قيمة.
- **العودة من دوال Lambda**: يمكن استخدام "return" أيضًا في دوال Lambda، لكن يجب الانتباه إلى أن "return" في Lambda تعود إلى الدالة المشتقة منها، وليس إلى Lambda نفسها.

### التفاصيل
- إذا لم يتم تحديد قيمة للإرجاع، يتم افتراض أن القيمة التي تُرجع هي `Unit`، والتي تعادل `void` في لغات برمجة أخرى.
- في حالة استخدام "return" داخل حلقة، فإنها ستؤدي إلى إنهاء الحلقة والدالة.

## الأمثلة
### مثال 1: دالة بسيطة ترجع قيمة
```kotlin
fun add(a: Int, b: Int): Int {
    return a + b
}
```
في هذا المثال، تقوم الدالة `add` بجمع عددين وإرجاع الناتج.

### مثال 2: دالة بدون قيمة إرجاع
```kotlin
fun printMessage(message: String) {
    println(message)
    return // يمكن استخدام return بدون قيمة
}
```
هنا، تقوم الدالة `printMessage` بطباعة رسالة، ويمكن إنهاؤها باستخدام "return" بدون قيمة.

### مثال 3: استخدام return في Lambda
```kotlin
val sum = { a: Int, b: Int ->
    return@sum a + b // هذا يعود إلى Lambda نفسها
}
```
في هذا المثال، "return@sum" تشير إلى العودة من Lambda.

## الشرح
### الأخطاء الشائعة
- **استخدام return في مكان غير مناسب**: بعض المبرمجين قد يخطئون في استخدام "return" داخل الحلقات أو الشرطيات دون فهم كيفية تأثير ذلك على تدفق البرنامج.
- **إرجاع قيم غير متوافقة**: يجب التأكد من أن القيمة المُرجعة تتطابق مع نوع الدالة المحدد.

### ملاحظات إضافية
- يمكن استخدام "return" مع الدوال ذات السلاسل والتعبيرات الشرطية، مما يجعل كوتلن لغة مرنة وسهلة الاستخدام.
- من الجيد استخدام "return" بطريقة منطقية، مما يساعد في تحسين وضوح الكود.

## ملخص من سطر واحد
تُستخدم الكلمة المفتاحية "return" في كوتلن لإنهاء الدالة وإرجاع قيمة، مما يسهل كتابة برامج مرتبة وواضحة.
