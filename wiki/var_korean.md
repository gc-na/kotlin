# Kotlin 변수 선언: var의 사용법과 특징

## 개요
Kotlin에서 `var`는 변수 선언을 위한 키워드로, 변경 가능한(mutable) 변수를 생성할 때 사용됩니다. 이는 Kotlin의 타입 추론 기능과 결합되어 코드의 가독성을 높이고, 개발자가 변수를 쉽게 관리할 수 있도록 도와줍니다.

## 문서화
Kotlin에서 `var`를 사용하여 변수를 선언할 수 있습니다. `var`로 선언된 변수는 나중에 재할당이 가능하여, 프로그램 실행 중 값이 변경될 수 있는 경우에 유용합니다. 변수의 타입은 명시적으로 지정할 수도 있고, 타입 추론을 통해 자동으로 결정할 수도 있습니다.

### 사용법
`var` 키워드를 사용하여 변수를 선언하는 기본 문법은 다음과 같습니다:

```kotlin
var 변수이름: 타입 = 초기값
```

타입을 생략하면 Kotlin이 초기값을 바탕으로 타입을 추론합니다.

### 예시
다음은 `var`를 사용한 기본적인 변수 선언 예시입니다.

```kotlin
// 타입을 명시적으로 지정
var age: Int = 25

// 타입을 추론
var name = "홍길동"

// 초기값 변경
age = 30
name = "김철수"
```

## 설명
`var`로 선언된 변수는 변경 가능하다는 특징이 있지만, 변수가 참조하는 객체는 immutable 할 수 있습니다. 예를 들어, `var`로 선언된 리스트는 리스트 자체를 변경할 수 있지만, 리스트의 항목은 불변으로 선언할 수 있습니다.

### 일반적인 실수 및 주의 사항
1. **타입 혼동**: `var`로 선언된 변수가 다른 타입으로 재할당되면 컴파일 오류가 발생합니다.
   ```kotlin
   var number = 10
   number = "문자열" // 오류 발생
   ```

2. **Null 안전**: 만약 변수가 null 값을 가질 수 있도록 하려면, 타입 뒤에 `?` 기호를 붙여야 합니다.
   ```kotlin
   var nullableString: String? = null
   ```

3. **가독성**: 변수 이름은 의미 있게 지어야 하며, `var` 키워드의 사용은 코드의 가독성을 높이는 데 기여합니다. 예를 들어, `var temp = 100` 대신 `var temperature = 100`와 같이 사용하는 것이 좋습니다.

## 한 줄 요약
Kotlin에서 `var`는 변경 가능한 변수를 선언할 때 사용하는 키워드로, 타입 추론 기능을 통해 코드의 가독성을 높여줍니다.
