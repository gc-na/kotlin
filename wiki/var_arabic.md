# استخدام "var" في لغة كوتلن: المتغيرات القابلة للتغيير

## ملخص
كلمة "var" في لغة كوتلن تُستخدم لتعريف المتغيرات القابلة للتغيير، مما يسمح بتعديل قيمتها بعد إنشائها. تعتبر "var" أداة قوية لتطبيق البرمجة الديناميكية.

## التوثيق
### الغرض
تُستخدم "var" لتحديد المتغيرات القابلة للتغيير في كوتلن، وهي تُعد إحدى التعريفات الأساسية للمتغيرات في اللغة. يمكن أن تحتوي المتغيرات المعرفة بواسطة "var" على قيم يمكن تعديلها لاحقًا، مما يجعلها مثالية للاستخدام في الحالات التي تحتاج فيها إلى تحديث القيم.

### الاستخدام
لتعريف متغير باستخدام "var"، يُستخدم الصيغة التالية:
```kotlin
var اسم_المتغير: نوع_البيانات = القيمة
```
حيث:
- `اسم_المتغير` هو الاسم الذي ستستخدمه للإشارة إلى المتغير.
- `نوع_البيانات` هو نوع البيانات (مثل Int، String، إلخ).
- `القيمة` هي القيمة الابتدائية للمتغير.

### التفاصيل
- المتغيرات التي يتم تعريفها باستخدام "var" يمكن تعديلها في أي وقت.
- يتم تخزين المتغيرات في الذاكرة، ويمكن الوصول إليها واستخدامها في مختلف أجزاء البرنامج.

## الأمثلة
### مثال 1: تعريف متغير عددي
```kotlin
var عدد: Int = 10
عدد = 15 // تعديل القيمة
println(عدد) // الناتج: 15
```

### مثال 2: تعريف متغير نصي
```kotlin
var اسم: String = "أحمد"
اسم = "فاطمة" // تعديل القيمة
println(اسم) // الناتج: فاطمة
```

### مثال 3: استخدام المتغير في دالة
```kotlin
fun طباعةعدد() {
    var عدد: Int = 5
    println("العدد هو: $عدد")
    عدد += 10 // تعديل القيمة
    println("العدد بعد التعديل هو: $عدد")
}

طباعةعدد()
// الناتج:
// العدد هو: 5
// العدد بعد التعديل هو: 15
```

## الشرح
### الأخطاء الشائعة
- **إعادة تعيين نوع البيانات**: لا يمكن تغيير نوع البيانات لمتغير تم تعريفه بـ "var" بعد إنشائه. فمثلاً، لا يمكن تعيين قيمة نصية لمتغير تم تعريفه كعدد صحيح.
- **عدم تهيئة المتغير**: يجب أن يتم تهيئة المتغير عند تعريفه؛ إلا إذا كان نوع البيانات قابلًا للاستخدام بدون قيمة (مثل Nullable types).

### ملاحظات إضافية
- يُفضل استخدام "val" لتعريف المتغيرات الثابتة التي لا تحتاج إلى تغيير، مما يزيد من وضوح الكود.
- "var" مثالية في الحالات التي تحتاج فيها إلى متغيرات تتغير بشكل ديناميكي مثل العدادات أو المتغيرات المستخدمة في حلقات التكرار.

## ملخص بجملة واحدة
تُستخدم "var" في كوتلن لتعريف المتغيرات القابلة للتغيير، مما يسمح بتعديل قيمتها بعد إنشائها.
