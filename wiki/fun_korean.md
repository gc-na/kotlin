# Kotlin의 fun 키워드: 함수 정의의 핵심

## 개요
Kotlin에서 `fun` 키워드는 함수를 정의하는 데 사용되는 기본적인 키워드입니다. 이 키워드는 코드의 재사용성을 높이고, 프로그램의 구조를 명확하게 하며, 기능적 프로그래밍 스타일을 지원합니다.

## 문서화
`fun` 키워드는 Kotlin에서 함수를 선언하는 데 사용됩니다. 함수는 특정 작업을 수행하고, 입력 값을 받아들이며, 결과를 반환할 수 있는 코드 블록입니다. Kotlin에서 함수는 전역적으로 또는 클래스 내부에서 정의될 수 있으며, 고차 함수, 익명 함수와 같은 다양한 형태로 활용될 수 있습니다.

### 사용법
기본적인 함수 정의는 다음과 같은 형식을 따릅니다:

```kotlin
fun 함수이름(매개변수이름: 매개변수타입): 반환타입 {
    // 함수 본체
}
```

- `함수이름`: 함수의 이름
- `매개변수이름`: 함수가 받을 인자의 이름
- `매개변수타입`: 각 매개변수의 데이터 타입
- `반환타입`: 함수가 반환하는 값의 타입 (없으면 `Unit`으로 간주됨)

## 예제
아래는 Kotlin에서 `fun` 키워드를 사용하여 함수를 정의하고 호출하는 기본적인 예제입니다.

```kotlin
fun add(a: Int, b: Int): Int {
    return a + b
}

fun main() {
    val result = add(5, 3)
    println("결과는 $result 입니다.") // 결과는 8 입니다.
}
```

위의 코드에서 `add` 함수는 두 개의 정수를 매개변수로 받아 그 합을 반환합니다.

## 설명
Kotlin의 `fun` 키워드를 사용할 때 주의해야 할 몇 가지 사항이 있습니다:

1. **타입 선언**: Kotlin은 강타입 언어이므로, 매개변수와 반환 타입을 명확하게 선언해야 합니다. 그렇지 않으면 컴파일 에러가 발생합니다.
   
2. **Unit 반환**: 반환 타입이 명시되지 않으면 기본적으로 `Unit`을 반환합니다. 이는 Java의 `void`와 유사합니다. 반환 값이 없더라도 `return` 문을 사용하면 안 됩니다.

3. **고차 함수**: Kotlin에서는 함수를 매개변수로 받을 수 있습니다. 이를 통해 함수형 프로그래밍 스타일을 적용할 수 있습니다. 

4. **람다 표현식**: 익명 함수나 람다 표현식으로도 함수를 정의할 수 있어 코드의 간결성을 높일 수 있습니다.

## 한 줄 요약
Kotlin의 `fun` 키워드는 함수 정의를 위한 필수 키워드로, 프로그램의 구조화와 재사용성을 극대화합니다.
