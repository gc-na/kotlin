# Kotlin 中的 "do" 指令：用途與示例

## 概述
在 Kotlin 程式語言中，"do" 是一個控制結構，與 "while" 搭配使用，能夠執行一段程式碼，直到指定的條件為假。這個結構特別適合需要至少執行一次的情況。

## 文檔
### 目的
"do" 指令的主要目的是執行一段程式碼塊，然後檢查條件是否為真。如果條件為真，則重複執行該程式碼塊。這使得 "do...while" 循環比 "while" 循環更具彈性，因為它保證至少執行一次。

### 使用方法
"do...while" 的基本語法如下：

```kotlin
do {
    // 要執行的程式碼
} while (條件)
```

在這個結構中：
- 程式碼塊中的內容將至少執行一次。
- 之後會檢查「條件」，如果條件為真，則會再次執行程式碼塊。

## 示例
### 基本使用範例

以下是一個簡單的範例，展示如何使用 "do...while" 循環：

```kotlin
var count = 1

do {
    println("當前計數: $count")
    count++
} while (count <= 5)
```

在此例中，程式將輸出計數從 1 到 5 的數字，因為 `do` 區塊至少會執行一次。

### 另一個範例

```kotlin
var input: Int

do {
    println("請輸入一個正整數:")
    input = readLine()!!.toInt()
} while (input <= 0)

println("您輸入的正整數是: $input")
```

這段程式碼會要求用戶輸入一個正整數，直到用戶輸入的數字滿足條件。

## 解釋
### 常見的陷阱和注意事項
- **條件必須在循環後檢查**：由於 "do" 循環的特性，不要期望循環在第一次檢查條件前停止。
- **避免無限循環**：確保在循環內部有邏輯能改變條件，否則可能會導致無限循環。
- **使用 `!!` 確保不為空**：在讀取用戶輸入時，使用 `!!` 來保證讀取的結果不為空，這樣可以避免 `NullPointerException`。

## 一句總結
在 Kotlin 中，"do" 指令用於保證至少執行一次的循環結構，與 "while" 搭配使用進行條件檢查。
