# استخدام جملة "while" في لغة كوتلن

## ملخص
تعتبر جملة "while" في كوتلن طريقة فعالة لتنفيذ حلقات التكرار، حيث تستمر الحلقة في التنفيذ طالما أن الشرط المحدد صحيح. توفر هذه الجملة تحكمًا مرنًا في تدفق البرنامج.

## الوثائق
تستخدم جملة "while" لتنفيذ كود معين عدة مرات حتى يتحقق شرط محدد. تتكون الجملة من عبارة شرطية، وإذا كانت هذه العبارة صحيحة، يتم تنفيذ كتلة التعليمات البرمجية داخل الحلقة. إذا كان الشرط غير صحيح، يتم الخروج من الحلقة.

### الصياغة:
```kotlin
while (شرط) {
    // تنفيذ التعليمات
}
```

### الاستخدام:
- **الشرط**: هو تعبير يُقيم إلى قيمة منطقية (Boolean) ويمكن أن يتغير خلال تكرار الحلقة.
- **كتلة التعليمات**: هي مجموعة من التعليمات التي تُنفذ طالما أن الشرط صحيح.

## أمثلة
### مثال 1: عد تنازلي باستخدام while
```kotlin
var count = 5
while (count > 0) {
    println("عدد العد: $count")
    count--
}
```
**النتيجة:**
```
عدد العد: 5
عدد العد: 4
عدد العد: 3
عدد العد: 2
عدد العد: 1
```

### مثال 2: قراءة الأعداد حتى يدخل المستخدم الصفر
```kotlin
var number: Int
do {
    println("أدخل رقم (أدخل 0 للخروج):")
    number = readLine()!!.toInt()
} while (number != 0)
```

## الشرح
### المفاهيم الشائعة:
- **عدم تحقيق الشرط**: إذا كان الشرط خاطئًا منذ البداية، فلن يتم تنفيذ كود الحلقة على الإطلاق.
- **تجنب الدورات اللانهائية**: تأكد من أن الشرط سيتغير في كل تكرار، وإلا فقد تتسبب في حلقة لانهائية.

### ملاحظات إضافية:
- يمكن استخدام جملة "while" بدلاً من جملة "for" في حالات معينة، خاصة عندما لا تعرف عدد التكرارات مسبقًا.
- يمكن استخدام جملة "do-while" كبديل، حيث يتم ضمان تنفيذ كتلة التعليمات مرة واحدة على الأقل.

## ملخص جملة واحدة
جملة "while" في كوتلن تسمح بتنفيذ كود معين بشكل متكرر طالما أن الشرط المحدد صحيح.
